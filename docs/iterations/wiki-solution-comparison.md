# 知识库方案对比与选型

> **总结**: 对比 Markdown, Notion, MediaWiki 等方案，最终选定 VitePress 作为个人知识库的技术栈，平衡了“内容持久性（Markdown）”与“阅读体验（现代网站）”的需求。

## 方案对比

### 1. 纯 Markdown + GitHub
- **优点**: 零成本、随时编辑、专注内容。
- **缺点**: 阅读体验平庸，缺乏高效搜索。

### 2. VitePress 网站 ⭐ 最终选定
- **优点**: 视觉精美、极速响应、内置全文搜索、完美适配移动端、保持 Markdown 编写。

### 3. Notion / 语雀
- **优点**: 可视化能力强。
- **缺点**: 依赖第三方，数据导出与自定义受限。

## 决策建议

**结论**: VitePress 是追求极致阅读体验与数据掌控力的最佳平衡点。
