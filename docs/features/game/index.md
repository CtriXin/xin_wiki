## 🎯 游戏玩法 (三大模式)

### 1. 狙击模式 (均衡型)
- **玩法**: 看题 -> 输入答案 -> 爆发连射。最纯粹的计算练习。

### 2. 冲撞模式 (速度型)
- **玩法**: 直接去撞击结果正确的方块。考验反应与心算。

### 3. 数字合成模式 (火力型)
- **玩法**: 手动射击方块累加数字，在限定的射击次数内，精准达到目标值。
- **特性**: 系统保证场上一定存在解，考验策略与心算。

---

## 📱 操作方式

- **移动**: 全屏拖拽飞机，精准灵活
- **射击**: 
  - 狙击模式: 输入正确答案后自动发射
  - 21点模式: 点击左下角 **FIRE** 按钮手动发射
- **大招**: 点击右侧 **ULT** 按钮全屏清场 (需充能)

---

## 🛠 技术实现

- **引擎**: Matter.js 物理引擎 (引入冲击波与物理感应)
- **渲染**: 多层星空背景 + 动态倾斜动画
- **架构**: 极简 HUD 界面，零视线遮挡
