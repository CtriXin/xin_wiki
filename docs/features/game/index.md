## 🎯 游戏玩法

1. **算出答案** - 解决数学题目
2. **输入答案** - 点击底部数字键盘
3. **自动发射** - 答对后发射对应数量的子弹
4. **消除方块** - 击落所有方块过关
5. **保护生命** - 漏掉 3 个方块游戏结束

---

## 🔥 狂暴模式 (Phase 1)

### 连击 x3 - 加速模式
- 子弹速度 **x1.5** (8 → 12)
- 发射间隔缩短 **50%** (80ms → 40ms)
- 屏幕轻微震动效果
- 飞机闪烁红光

### 连击 x5 - 穿透模式
- 子弹可穿透 **2 个方块**
- 子弹变成**金色**并发光
- 拖尾加长

### 答案 ≥ 20 - 核弹清屏
- 屏幕**白光一闪**
- 清除中心 **20%** 区域方块
- 播放胜利号角音效
- 金币爆射特效

## 📱 操作方式

- **移动**: 点击左右半屏 / 开启自动瞄准
- **输入**: 底部数字键盘 (0-9 + 清除)
- **切换瞄准**: 点击顶部"瞄准"按钮
- **音效**: 点击顶部"🔊"开关

---

## 🛠 技术实现

- **引擎**: Matter.js 物理引擎
- **渲染**: HTML5 Canvas
- **音效**: Web Audio API
- **架构**: 单页面应用，响应式布局
