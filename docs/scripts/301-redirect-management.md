# 301 重定向管理 (ptc_301)

> **总结**: 整理并记录 `ptc_301` 项目的快捷维护指令，实现对服务器 301 重定向规则的快速添加、列表查询及配置提交。

## 快捷命令速查

| 命令 | 说明 |
|------|------|
| `301` | 进入 301 管理目录并启动交互式 CLI |
| `301add` | 添加新的重定向规则 |
| `301list` | 列出当前所有重定向规则 |
| `301commit` | 提交并应用当前更改 |

## 常用操作流程

### 1. 添加规则
```bash
301add
# 按照提示输入源路径和目标 URL
```

### 2. 提交生效
```bash
301commit
```
