## 背景与需求

项目 `ptc_301` 用于管理服务器的 301 重定向规则。为了简化操作，封装了一系列快捷命令来处理规则的添加、删除、恢复和提交。

## 快捷命令速查

| 命令 | 说明 |
|------|------|
| `301` | 进入 301 管理目录并启动交互式 CLI |
| `301add` | 添加新的重定向规则 |
| `301del` | 删除指定的重定向规则 |
| `301restore` | 恢复到之前的配置版本 |
| `301commit` | 提交并应用当前更改 |
| `301list` | 列出当前所有重定向规则 |
| `301batch` | 执行批量规则操作 |

## 常用操作流程

### 1. 添加规则
```bash
301add
# 按照提示输入源路径和目标 URL
```

### 2. 查看与确认
```bash
301list
```

### 3. 提交生效
```bash
301commit
```
